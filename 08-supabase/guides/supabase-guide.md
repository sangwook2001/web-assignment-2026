# Supabase 가이드

이 문서는 학생이 Supabase 웹사이트에서 직접 프로젝트를 만들고, 테이블을 생성하고, JavaScript 연결에 필요한 값을 어디서 확인하는지 한 번에 따라갈 수 있도록 정리한 안내서입니다.

## 1. Supabase 프로젝트 만들기

1. `https://supabase.com`에 접속해 로그인합니다.
2. `New project`를 클릭합니다.
3. 조직을 선택합니다.
4. 아래 항목을 입력합니다.
   - 프로젝트 이름
   - 데이터베이스 비밀번호
   - 리전
5. `Create new project`를 클릭합니다.
6. 프로젝트 대시보드가 준비될 때까지 기다립니다.

## 2. `todo` 테이블 만들기

테이블은 두 가지 방법으로 만들 수 있습니다.

### 방법 A. Table Editor 사용

1. `Table Editor`를 엽니다.
2. `Create a new table`를 클릭합니다.
3. 테이블 이름을 `todo`로 입력합니다.
4. 아래 컬럼을 추가합니다.

| 컬럼명 | 타입 | 설명 |
| --- | --- | --- |
| `id` | `int8` | 기본 키, identity |
| `content` | `text` | not null |
| `is_done` | `bool` | 기본값 `false` |
| `created_at` | `timestamptz` | 기본값 `now()` |

5. 저장합니다.

### 방법 B. SQL Editor 사용

1. `SQL Editor`를 엽니다.
2. `New query`를 클릭합니다.
3. 아래 SQL을 실행합니다.

```sql
create table if not exists public.todo (
  id bigint generated by default as identity primary key,
  content text not null,
  is_done boolean not null default false,
  created_at timestamptz not null default now()
);
```

## 3. API 연결에 필요한 값 찾기

JavaScript 코드에는 아래 두 값이 필요합니다.

- `Project URL`
- `anon public key`

확인 경로는 다음과 같습니다.

1. Supabase 프로젝트를 엽니다.
2. `Settings`로 이동합니다.
3. `API`를 엽니다.
4. 아래 값을 복사합니다.
   - `Project URL`
   - `anon` 또는 `anon public` 키

예시는 아래와 같습니다.

```js
const supabaseUrl = "https://your-project-id.supabase.co";
const supabaseKey = "your-anon-public-key";
```

## 4. 어떤 키를 사용해야 하나

- 프론트엔드 실습 코드에는 `anon public key`를 사용합니다.
- `service_role` 키는 서버 전용입니다.
- 브라우저 JavaScript에 `service_role` 키를 넣으면 안 됩니다.

학생이 가장 자주 하는 실수 중 하나가 `anon` 키 대신 `service_role` 키를 복사하는 것입니다. 수업 자료에서 이 점을 분명하게 안내해야 합니다.

## 5. 이름은 코드와 정확히 일치해야 한다

코드와 Supabase 테이블의 이름은 완전히 같아야 합니다.

- 테이블 이름: `todo`
- 컬럼 이름: `id`, `content`, `is_done`, `created_at`

예를 들어 코드에 아래처럼 적혀 있다면

```js
.from("todo")
.select("id, content, is_done, created_at")
```

Supabase에서도 같은 이름으로 만들어야 합니다.

## 6. 학생이 자주 막히는 지점

- 프로젝트는 만들었지만 테이블은 아직 만들지 않은 경우
- 테이블 이름이 `todo`가 아닌 경우
- `is_done` 대신 `isDone`처럼 다른 이름으로 만든 경우
- `service_role` 키를 복사한 경우
- URL이나 key 앞뒤에 공백이 들어간 경우
